> [!definition|*] Set Equality
> 
> Sets $A$ and $B$ are equal, denoted $A=B$, if and only if:
> 1. $A \subseteq B$
> 2. $B \subseteq A$
> 
> Note that the $=$ sign is a nonnumeric equal sign.

# Direct of Proof $A = B$
To show sets $A$ and $B$ are equal, we:
1. Show $A \subseteq B$
2. Show $B \subseteq A$
3. Conclude $A = B$ by definition (mic drop!)

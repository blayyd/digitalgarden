> [!definition|*] Set of All Inversions for a Given $\pi \in S_{n}$
> 
> Let $\pi \in S_{n}$ be given. Then, let the **set of all inversions with respect to** $\pi$ be
> $$
> \text{Inv}(\pi) = \{ (i,j): i,j \in [n], i<j, \text{ and } \pi(i)>\pi(j) \}
> $$

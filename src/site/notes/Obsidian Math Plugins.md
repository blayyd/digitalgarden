# CSS Snippets
- Bigger math font
```css
mjx-math {
  font-size: 120% !important;
}
```

- Center images
```css
img {
    display: block;
    margin-left: auto;
    margin-right: auto;
}

/* === Images size defaults === */
.image-embed.image-embed img:not([width]) {
    max-height: 450px;
    max-width: 500px;
}


```
- Center Desmos graphs
```css
/* Horizontally center the graph in the page content */
.desmos-graph {
    display: block;
    margin-left: auto;
    margin-right: auto;
}
```
